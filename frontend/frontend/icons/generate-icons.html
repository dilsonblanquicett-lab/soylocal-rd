Instrucciones para crear íconos PWA

Para una PWA completa, necesitas imágenes reales. Mientras tanto, puedes usar este emoji como ícono temporal:

🏖️

Emoji beach

Crear íconos reales (recomendado):

  1. Ve a favicon-generator.org
  2. Sube una imagen (puedes usar el logo de SoyLocal RD)
  3. Descarga los íconos generados
  4. Copia los archivos a la carpeta frontend/icons/

Alternativa rápida:

Usa este comando en PowerShell para crear íconos simples (requiere PowerShell 7+):

cd frontend/icons; 72..512 | ForEach-Object { New-Item -Path "icon-${_}x${_}.png" -ItemType File -Force }
